# Changelog

## [0.1.0](https://github.com/Texarkanine/jekyll-mermaid-prebuild/releases/tag/v0.1.0) (2026-01-17)

### Features

* Initial release
* Convert mermaid code blocks to SVG at build time
* Support backtick and tilde fenced code blocks
* Content-based caching for fast rebuilds
* Configurable output directory
* Clickable diagrams (link to full-size SVG)
* Helpful error messages for Puppeteer dependencies
